# 离线地图应用

这是一个基于 Leaflet.js 和 OpenStreetMap 的离线地图应用。该应用允许用户在没有网络连接的情况下查看已缓存的地图区域。

## 功能特点

- 使用 OpenStreetMap 作为地图数据源
- 支持离线模式切换
- 自动缓存已加载的地图瓦片
- 响应式设计，适配各种屏幕尺寸

## 使用方法

1. 打开 `index.html` 文件
2. 浏览需要离线使用的地图区域
3. 点击右上角的"离线模式"按钮切换到离线模式
4. 在离线模式下，只能查看已缓存的区域

## 技术栈

- Leaflet.js
- OpenStreetMap
- HTML5
- CSS3
- JavaScript (ES6+)

## 注意事项

- 首次使用时需要联网加载地图数据
- 离线模式下只能查看已缓存的区域
- 建议在需要离线使用的区域预先加载并缓存地图数据
