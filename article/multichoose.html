在 VSCode 中，你可以通过 多光标编辑 功能实现选中不同区域的文本块并进行批量修改。以下是具体步骤： 方法 1：使用鼠标 + Alt 键 选中第一个文本块：用鼠标拖动选择第一个文本块。 添加多个光标： 按住 Alt 键（Windows/Linux）或 Option 键（Mac）。 用鼠标拖动选择其他文本块，每选择一个区域都会添加一个新的光标。 批量修改：直接输入或修改内容，所有光标处的内容会同步更新。 方法 2：使用 Ctrl + Alt + 方向键（Windows/Linux）或 Cmd + Option + 方向键（Mac） 选中第一个文本块：用鼠标拖动选择第一个文本块。 添加多个光标： 按住 Ctrl + Alt（Windows/Linux）或 Cmd + Option（Mac）。 按 ↑ 或 ↓ 方向键，在相同列位置添加光标。 批量修改：直接输入或修改内容，所有光标处的内容会同步更新。 方法 3：使用 Ctrl + D（Windows/Linux）或 Cmd + D（Mac） 选中第一个文本块：用鼠标拖动选择第一个文本块。 添加多个光标： 按 Ctrl + D（Windows/Linux）或 Cmd + D（Mac）选择下一个相同的文本块。 重复按 Ctrl + D / Cmd + D 选择更多相同的文本块。 批量修改：直接输入或修改内容，所有光标处的内容会同步更新。 方法 4：使用 Shift + Alt + 鼠标拖动（列选择模式） 进入列选择模式： 按住 Shift + Alt（Windows/Linux）或 Shift + Option（Mac）。 用鼠标拖动选择多列文本。 批量修改：直接输入或修改内容，所有选中列的内容会同步更新。 方法 5：使用正则表达式替换 打开替换功能：按 Ctrl + H（Windows/Linux）或 Cmd + Option + F（Mac）。 启用正则表达式：点击替换框右侧的正则表达式图标（.*）。 输入正则表达式：编写匹配目标文本的正则表达式。 替换内容：输入替换内容，点击“全部替换”按钮。
